# Rinon
Electon Browser

# Usage
1. `$ git clone git@github.com:kohei-takata/Rinon.git`
2. `$ npm install`
3. Put `/Users/<User Name>/Library/Application Support/Rinon` `menu.json`
4. `$ cd Rinon`
5. `$ electron .`

`menu.json` looks like this.

```json
{
  "main" : "https://github.com/",
  "sub" : [
    "https://google.com/",
    "https://yahoo.com/"
  ]
}

```
