# Rinon
Electon Browser

# Usage
1. `$ git clone git@github.com:kohei-takata/Rinon.git`
2. `$ cd Rinon`
3. `$ npm install`
4. Put`menu.json` to `/Users/<User Name>/Library/Application Support/Rinon`
5. `$ electron .`

`menu.json` looks like this.

```json
{
  "main" : "https://github.com/",
  "sub" : [
    "https://github.com/",
    "https://google.com/",
    "https://yahoo.com/"
  ]
}

```
